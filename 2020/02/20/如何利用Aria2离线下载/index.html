如何利用Aria2离线下载
发布于：2020-02-20 21:08:33 标签：/ Aria2 / 网盘 / 访问：

桌面端

1.AriaNg

AriaNg是一款方便简洁的aria2下载Web客户端，你可以在自己的VPS上搭建，也可以使用别人搭建好的页面。最简捷的办法是使用 Aria2 For Chrome 扩展程序，下面将讲解如何使用这款插件进行离线下载的操作。

在Chrome应用商店里安装 Aria2 For Chorme，打开后在AriaNg设置中点击加号新建RPC。

在Aria2 RPC地址栏中填入VPS的IP地址，在RPC密钥栏中填入口令，点击右上角的重新加载按钮，就会成功连接到你的VPS进行离线下载了。

2.PanDownload

PanDownload是一款强大的网盘下载软件，使用需要登录百度账号（用多了会被限速，SVIP也限）。在 设置-远程-添加 中填入VPS的IP地址、端口号（默认6800）、口令，测试连接成功后保存。下载文件时，设置下载存储路径为之前添加的远程路径。

使用PanDownload进行BT下载的方式是先离线下载到百度网盘，再用上述方法下载转存。

移动端

安装Aria2APP，设置方法与桌面端大同小异。
